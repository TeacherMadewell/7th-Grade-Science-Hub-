Core procedures — we practice each one right now

  1. ENTER: come in, pick up the handout, start Bell Work — no reminder needed.
  2. ATTENTION SIGNAL: I say "Scientists!" → you say "Ready!", eyes up, hands empty, within 5 seconds.
  3. VOICE LEVELS: 0 silent · 1 partner whisper · 2 table talk · 3 presenting · 4 outside. I always name the level.
  4. MATERIALS: one person per table gets and returns supplies from the lab cart by table number.
  5. HELP: ask your table first, then raise your hand AND keep working.
  6. DISMISSAL: the bell doesn't dismiss you — I do, after the room is reset.

Practice it now — we run each one until it's automatic.
